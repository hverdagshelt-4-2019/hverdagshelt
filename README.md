# Hverdagshelt client and server application using react and express
The application will reload on changes.

## Server: start

From the top-level repository folder:
```sh
cd server
npm install
npm start
```

## Client: start

From the top-level repository folder:
```sh
cd client
npm install
npm start
```

## Tests: client
From the top-level repository folder:
Start server before you run client and tests
```sh
cd client
npm install
npm test
```
